# Javi's Blog

This is the personal blog of Javi Manzano.

## How to build the thing

- Create a virtualenv and activate it
    - `python3 -m venv env`
    - `source ./env/bin/activate`
- Install the dependencies
    - `pip install -r requirements.txt`
    - `npm install`
- Generate the blog
    - `python generate_blog.py`
- Serve it locally
    - `gulp`
